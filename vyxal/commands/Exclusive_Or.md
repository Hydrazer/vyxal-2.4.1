# `Ẋ` - Exclusive Or (Logical)

- Arity: 2
- In: a, b
- Out: a xor b

Returns a xor b. Preferred over `a b≠` because of the fact that it doesn't require values to be 1 or 0.

# Usage
```
0 1Ẋ║⟨1⟩
1 1Ẋ║⟨0⟩
0 0Ẋ║⟨0⟩
```

# Alternate Representations
_These require `a` and `b` to be either 0 or 1_
```
≠
=0=
```
